AQ Data Pulse — Accessibility

AQ Data Pulse is designed to work with Apple’s built-in accessibility features on iPhone and iPad.

Supported

  • VoiceOver — Dashboard metrics, health score, trend chart, alerts, and status banners include spoken labels and values. Alert filters announce selected state.
  • Dark Mode — The app follows the system light and dark appearance.
  • Differentiate Without Color — Status and severity use text labels, not color alone.
  • Sufficient Contrast — Standard system colors and typography are used throughout.
  • Reduce Motion — Health score animation and alert transitions respect the Reduce Motion setting.

Partial support

  • Larger Text (Dynamic Type) — Key text uses scalable system fonts; some layouts may not expand fully at the largest accessibility sizes.
